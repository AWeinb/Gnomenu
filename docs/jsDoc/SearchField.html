Class: SearchField

SearchField

This class creates the search entry at the top. The basic principle behind it is same as that of the normal Gnome search. When text is typed into the box or a textevent reaches the component it is send to the searchsystem, which then computes the results. The component is dependent on model update events which tell it when new results are available. Most functionallity is handled inside the class and the only thing needed is keyfocus. But nether the less the class provides some methods to interact with the component.
