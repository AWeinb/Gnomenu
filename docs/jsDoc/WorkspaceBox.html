Class: WorkspaceBox

WorkspaceBox

This is the component which contains the workspace views. It provides mouse and keyboard controls. To integrate drag events it needs to get this events from the buttons. Luckily the buttons provide this information and the mediator sends it further to the components.

Constructor

new WorkspaceBox(model, mediator)

For this class to work properly you need to provide a valid mediator instance. The constructor then creates the actor and starts listening to changes in the workspaces.
