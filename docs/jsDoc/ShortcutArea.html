Class: ShortcutArea

ShortcutArea

This component uses the list or gridview elements to create the part of the menu that shows the apps. This class automatically updates itself if the model was correctly provided. In that case the update events trigger an automatic update of the component. There are some methods implemented to control this component with the keyboard. Also it is simple to change the viewmode from list to grid and viseversa. It also should be no problem to add more viewmodes.
