Class: ShortcutBoxBase

ShortcutBoxBase

This is the base class for all shortcut boxes. It provides the basic methods that every box probably needs. It defines the basic structures and also implements big parts of the keyboard controls.

Constructor

new ShortcutBoxBase(mediator)

Creates basic functions which are useful for every kind of box. It does not create an actor.
