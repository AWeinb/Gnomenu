Class: PlacesManager

PlacesManager

This class keeps track of the places and devices. It notifies about changes via a callback function. You are not supposed to use this class but to use the Datamanager which provides signals as notifications.

Constructor

new PlacesManager(useSymbolic, cbDevices, cbNetDevices, cbBookmarks)

The constructor takes callback functions to notify about changes.
