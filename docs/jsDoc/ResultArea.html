Class: ResultArea

ResultArea

This creates the result area which is shown while a search is active. It is an element that is build onto a scrollview and can be added to a normal boxlayout. The class provides some methods to change the viewmode or to handle keyboard input. The element is at this moment not ment to stay as top component in the menu but that would be possible without major changes. The task of this is to display the searchresults of the searchproviders ordered by provider in the current viewmode. This class depends on searchupdates. You need to connect the "updateSearch" method correctly. The results are then taken directly out of the model. It is not needed to provide them directly.
