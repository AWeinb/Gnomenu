Class: Button

Button

This is the base class for all buttons. It provides all basic functionallity that can be fitted here without much more complexity. It does not need a model or mediator to work and should be extended for the use. Many of this methods are used in the subclasses in the menubutton module. Some of the private callback methods here are only marginally implemented and need to be implemented by the subclass. For example the _notify methods. The constructor should not fail in case you provide no icon and label. But it can fail if the icon is none of now used types. Some methods need an integer button argument. The enum EMousebutton contains some useful information for this.
