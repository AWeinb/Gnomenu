Class: PanelButton

PanelButton

This class is a simple button that can be used in the main panel. It provides methods to change text, icon and buttonhandler. Also you can use a hotspot at top border of the button to activate it. Besides that you can add keyboard shortcuts to activate the button.

Constructor

new PanelButton()

You should provide a name or an icon. If not you wont get a big button.
